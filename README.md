Student ManagementSystem-Java Console Based Application:

This is a simple Java console-based Student Management System designed for beginners to understand core Java concepts like classes, objects, methods, ArrayList, and CRUD operations.
The application allows users to add, view, find, update, and delete students through a menu-driven interface.

Features

-->Add a new student
-->View all students
-->Find a student by ID
-->Update existing student details
-->Delete a student by ID
-->Stores data using ArrayList (in-memory storage)
-->Simple, clean console interface
